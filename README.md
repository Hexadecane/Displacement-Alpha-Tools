# Displacement-Alpha-Tools
Application for converting Source Engine VMF displacement alpha painting to images, and vice-versa.

Created by Hxdce.
## Basic Build Instructions

1. Clone this repo.
2. Open the .sln file with Visual Studio 2019 (Not tested with earlier verions).
3. Compile it. 
4. Run it: `./bin/Release/DisplacementAlphaTools.exe` or `./bin/Debug/DisplacementAlphaTools.exe`
